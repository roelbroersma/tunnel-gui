* Tunneling Device
Flask GUI

** How to setup and run
Make sure you're in the project directory.
#+begin_src shell
  pip install -r requirements.txt
  flask run
#+end_src

** Install via the install script
If you want to install without git or any additional command, simply run the following one-liner.
#+begin_src shell
  wget -q -O - https://github.com/roelbroersma/tunneling-gui/raw/main/install.txt | bash
#+end_src
